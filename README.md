# Cleanmate – Premium Cleaning Service Homepage

A high‑fidelity, dependency‑free HTML/CSS/JS website for Cleanmate – a modern cleaning service brand. Built entirely from scratch without Tailwind, Bootstrap, jQuery, or page builders. This project showcases advanced front‑end engineering, custom interactivity, and pixel‑perfect UI replication from a premium WordPress theme.

## ✨ Key Features & Deliverables

### 🎨 UI/UX & Design
- **Pixel‑perfect replica** of a premium WordPress theme using only screenshots as reference.
- **Brand typography** (Raleway) identified and implemented strictly from visual assets.
- **Strategic UI/UX auditing** – full UX, conversion, and SEO audit of an enterprise consulting website (Phillips Consulting), leading to a modern redesign strategy and B2B sales playbook.

### 🧱 Advanced CSS Layouts
- **Bulletproof flattened grid** – asymmetrical image gallery (two stacked images seamlessly next to one full‑height image) without layout distortion or table hacks.
- **Complex 3‑column service card layout** with perfect alignment using CSS Grid + Flexbox.
- **Whitespace control** – wrapper containers with `max-width` and `margin: 0 auto` for global symmetry.
- **Responsive, mobile‑first** – grids reflow flawlessly from desktop to 375px.

### ⚡ Vanilla JavaScript Interactivity
- **Interactive pricing calculator** – real‑time math engine with range slider (square footage) + frequency dropdown (discounts). Zero server calls, instant DOM updates.
- **Click‑and‑drag (swipe) carousel** – custom mouse tracking (`mousedown`, `mousemove`, `mouseup`) with dynamic pagination dots generated by viewport width.
- **Custom lightbox modal** – image gallery with background scroll‑locking, Escape‑key close, overlay click‑to‑close, and ARIA accessibility.
- **State‑driven text swapping** – “Priorities” section using `data-text` attributes and `setTimeout` to fade/swap paragraph content on icon click.

### 🎞️ Scroll & Motion Effects
- **GSAP + ScrollTrigger** – smooth fade‑ups, directional slide‑ins, and animated number counters that trigger on scroll.
- **Zero‑latency animations** – no jank, fully optimised for performance.

### 🛠️ Debugging & Code Quality
- Debugged fatal JS crash caused by removed DOM element.
- Traced major layout breakage to a single rogue `</div>` tag – restored UI instantly.
- **Dependency‑free** – pure HTML5, CSS3, ES6+. Blazing fast load times, superior SEO.
